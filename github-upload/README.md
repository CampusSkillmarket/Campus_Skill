## 📁 文件结构

每个技能包包含：
```
skill-name.zip
├── SKILL.md       # 主技能文件（AI 配置）
├── package.json   # npm 包配置
├── index.js       # Node.js 入口
├── README.md      # 使用说明
└── CHANGELOG.md   # 变更日志
```

## 🚀 使用方式

1. 下载 zip 文件
2. 解压：`unzip skill-name.zip`
3. 将 `SKILL.md` 内容复制到 AI 助手配置
4. 开始使用！
