## 📥 下载方式

### 方式一：在线查看 SKILL.md（推荐）

1. 点击 [SKILL.md](https://github.com/CampusSkillmarket/skillhub-skills/blob/main/skills/time-management/SKILL.md) 查看完整内容
2. 复制全部内容
3. 粘贴到你的 AI 助手配置中

### 方式二：下载 ZIP 文件

1. 点击下载 [time-management.zip](https://github.com/CampusSkillmarket/skillhub-skills/raw/main/skills/time-management/time-management.zip)
2. 解压后打开 `SKILL.md`
3. 复制内容到 AI 助手配置

### 方式三：克隆整个仓库

```bash

```
