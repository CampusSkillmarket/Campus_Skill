# 📝 Word 文档写作助手 (Word Document Writer)

> 从 GitHub 开源项目整理的专业 Word 文档写作 Skill

## 技能描述

专为大学生和职场人士设计的 Word 文档写作技能。基于 GitHub 上开源的优秀 prompt 模板整理而成，帮助用户高效完成各类文档写作任务，包括论文、报告、简历、商业计划书等。

## 核心能力

### 1. 学术论文写作
- 论文大纲生成
- 摘要写作
- 文献综述撰写
- 研究方法描述
- 结论与讨论写作

### 2. 商务文档写作
- 商业计划书
- 项目报告
- 会议纪要
- 商务邮件
- 提案文档

### 3. 求职文档写作
- 简历优化
- 求职信
- 自我介绍
- 推荐信请求

### 4. 创意写作
- 故事创作
- 剧本写作
- 营销文案
- 演讲稿

## Prompt 模板

### 论文写作模板

```
请帮我写一篇关于 [主题] 的 [论文类型]。

要求：
1. 字数：[字数要求]
2. 格式：[格式要求，如 APA/MLA/GB/T 7714]
3. 结构：[需要的章节]
4. 重点：[需要重点阐述的内容]

背景信息：
- 研究目的：[目的]
- 研究方法：[方法]
- 数据来源：[数据来源]

请按照以下结构生成：
1. 摘要（200-300 字）
2. 引言
3. 文献综述
4. 研究方法
5. 结果与分析
6. 结论
7. 参考文献
```


## 写作技巧

### 1. 清晰结构
- 使用标题和副标题
- 段落分明，每段一个中心思想
- 使用列表和表格增强可读性

### 2. 简洁表达
- 避免冗长句子
- 使用主动语态
- 删除无关内容

### 3. 数据支撑
- 用数据说明成果
- 引用权威来源
- 制作图表可视化

### 4. 专业格式
- 统一字体和字号
- 规范引用格式
- 检查拼写和语法

## 注意事项

1. **原创性**：生成的内容需要根据自己的情况修改，避免直接抄袭
2. **准确性**：数据和事实需要核实，确保准确无误
3. **格式规范**：按照学校或公司的格式要求调整
4. **多次修改**：好文章是改出来的，建议多次迭代优化
