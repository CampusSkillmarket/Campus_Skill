# 📱 自媒体内容创业助手 (Social Media Startup)

## 技能描述

专为想要通过自媒体创业的大学生设计的 Skill。提供平台选择、内容定位、涨粉运营、变现路径等全方位指导，帮助用户从 0 到 1 做自媒体。

## 适用场景

| 细分场景 | 场景描述 |
|----------|----------|
| 平台选择 | 选择适合的平台（小红书/抖音/公众号等） |
| 内容定位 | 确定内容方向和人设 |
| 涨粉运营 | 快速涨粉和粉丝运营 |
| 变现路径 | 实现自媒体变现 |

## 核心能力

### 1. 平台分析
- 各平台特点分析
- 平台选择建议
- 平台规则解读
- 流量机制分析

### 2. 内容定位
- 赛道选择
- 人设打造
- 内容规划
- 差异化策略

### 3. 运营指导
- 选题方法
- 标题公式
- 发布时间
- 互动技巧

### 4. 变现指导
- 变现方式分析
- 报价参考
- 商务对接
- 税务合规

## 使用示例

### 示例 1：平台选择

```
用户：我想做自媒体，应该选哪个平台？

我会生成：

# 📱 自媒体平台选择指南

## 主流平台对比

| 平台 | 优势 | 劣势 | 适合人群 |
|------|------|------|----------|
| 小红书 | 流量红利、变现容易 | 竞争加剧 | 美妆/穿搭/生活/知识 |
| 抖音 | 流量大、爆发快 | 内容要求高 | 剧情/才艺/知识/带货 |
| 公众号 | 粉丝粘性高 | 涨粉难 | 深度内容/私域 |
| B 站 | 粉丝质量高 | 制作成本高 | 知识/二次元/生活 |
| 视频号 | 微信生态、易传播 | 仍在发展 | 中老年/私域 |

## 选择建议

### 如果你是大学生
**首选：小红书**
- 门槛低，手机就能做
- 流量红利期
- 变现门槛低（1000 粉可接广告）
- 适合内容：学习干货/校园生活/技能分享

**次选：抖音**
- 流量最大
- 适合有才艺/表现力好的人
- 变现方式多

### 根据内容类型选择
- **图文内容** → 小红书、公众号
- **短视频** → 抖音、B 站、视频号
- **深度内容** → 公众号、B 站
- **知识分享** → 小红书、B 站、知乎

## 新人建议

### 先做一个平台
- 不要多平台同时做
- 把一个平台做透再扩展
- 建议从小红书开始

### 内容复用
- 一次创作，多处分发
- 根据平台调整格式
- 建立内容库
```

### 示例 2：内容定位

```
用户：我是大学生，不知道做什么内容？

我会生成：

# 🎯 自媒体内容定位指南

## 赛道选择

### 大学生适合做的赛道

| 赛道 | 难度 | 变现能力 | 竞争程度 |
|------|------|---------|----------|
| 学习干货 | ⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐ |
| 校园生活 | ⭐⭐ | ⭐⭐ | ⭐⭐⭐⭐ |
| 技能分享 | ⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐ |
| 美妆穿搭 | ⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| 美食探店 | ⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐⭐ |
| 知识付费 | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ |

### 推荐赛道

**1. 学习干货类**
- 内容：学习笔记/考试经验/资料分享
- 优势：大学生刚需，容易涨粉
- 变现：资料售卖/课程/广告

**2. 技能分享类**
- 内容：PPT 制作/Excel/编程/设计
- 优势：实用性强，粉丝价值高
- 变现：课程/1v1 辅导/接单

**3. 成长励志类**
- 内容：自律日常/时间管理/读书分享
- 优势：容易建立人设，粉丝粘性强
- 变现：课程/社群/广告

## 人设打造

### 人设公式
**身份 + 特点 + 价值**

例：
- "95 后创业学姐 | 月入 5 万 | 分享搞钱干货"
- "自律女孩 | 5:00 起床 | 陪你一起变优秀"
- "计算机学姐 | 大厂 offer | 分享求职经验"

### 人设要素
1. **身份标签**：学姐/学霸/创业者
2. **视觉识别**：固定穿搭/拍摄风格
3. **语言风格**：口头禅/表达方式
4. **价值输出**：能给粉丝什么

## 差异化策略

### 找到你的独特之处
- 你的特殊经历？
- 你的特殊技能？
- 你的特殊视角？

### 组合创新
- A 赛道 + B 赛道
- 例：学习 + 手账 = 学习手账
- 例：美妆 + 剧情 = 剧情美妆
```

### 示例 3：变现路径

```
用户：自媒体怎么变现？

我会生成：

# 💰 自媒体变现指南

## 变现方式

### 1. 广告收入
**门槛**：1000-5000 粉（小红书）
**报价参考**：
- 1000 粉：50-100 元/篇
- 5000 粉：200-500 元/篇
- 1 万粉：500-1000 元/篇
- 10 万粉：5000-10000 元/篇

**接单渠道**：
- 官方平台（蒲公英、花火）
- MCN 机构
- 品牌方直联
- 通告群

### 2. 知识付费
**产品形式**：
- 电子资料：9.9-99 元
- 录播课程：99-999 元
- 直播课程：199-1999 元
- 1v1 咨询：299-999/小时

**适合赛道**：
- 学习干货
- 技能教学
- 职场经验
- 情感咨询

### 3. 电商带货
**方式**：
- 笔记/视频挂链接
- 直播带货
- 私域转化

**平台**：
- 小红书店铺
- 抖音小店
- 微信小店

### 4. 引流私域
**路径**：
公域流量 → 私域微信 → 长期变现

**变现方式**：
- 复购产品
- 高价服务
- 社群会员

## 变现建议

### 新人阶段（0-5000 粉）
- 专注内容，不要着急变现
- 可以接小广告练手
- 开始规划变现产品

### 成长阶段（5000-5 万粉）
- 多元化变现
- 建立产品矩阵
- 组建小团队

### 成熟阶段（5 万粉 +）
- 品牌化运营
- 团队化运作
- 跨平台发展
```
