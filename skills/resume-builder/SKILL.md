# 📄 简历制作与优化助手 (Resume Builder)

## 技能描述

专为大学生和职场新人设计的简历制作 Skill。提供简历模板、内容优化、排版建议、ATS 优化等功能，帮助打造 HR 一眼看中的简历。

## 适用场景

| 细分场景 | 场景描述 |
|----------|----------|
| 零实习简历 | 没有实习经历，简历没内容 |
| 简历排版 | 需要设计简历格式和排版 |
| 针对性优化 | 针对不同岗位优化简历 |
| 简历诊断 | 需要别人帮忙看简历问题 |

## 核心能力

### 1. 简历模板提供
- 按行业分类的模板（技术/市场/金融等）
- 按经验分类的模板（实习/校招/社招）
- 中英文简历模板
- LaTeX/Word/PDF 格式

### 2. 内容优化
- STAR 法则改写经历
- 量化成果建议
- 关键词优化（ATS 友好）
- 动词使用建议

### 3. 排版建议
- 一页纸原则
- 信息层次分明
- 字体字号规范
- 留白适当

### 4. 简历诊断
- 内容完整性检查
- 格式规范性检查
- 关键词匹配度
- 改进建议

## 使用示例

### 示例 1：零实习简历

```
用户：我没有实习经历，简历怎么写？

我会生成：

# 📝 零实习简历制作指南

## 挖掘可用经历

### 校园经历
- 学生会/社团：组织过什么活动？
- 班级职务：做过什么服务？
- 志愿活动：参与过什么公益？

### 课程项目
- 小组作业：承担什么角色？
- 课程设计：做出什么作品？
- 竞赛经历：获得什么奖项？

### 个人作品
- GitHub 项目
- 设计作品
- 写作文章
- 自媒体账号

## 简历框架

### 基本信息
- 姓名、联系方式、邮箱
- 求职意向

### 教育背景
- 学校、专业、GPA、排名
- 核心课程、奖学金

### 项目经历（重点）
- 课程项目 1
- 课程项目 2
- 个人作品

### 校园经历
- 社团/学生会
- 志愿活动

### 技能证书
- 专业技能
- 语言能力
- 证书

## STAR 法则改写示例

**原句：**
"参与学生会组织的迎新活动"

**改写后：**
"作为迎新活动负责人（S），负责 200 名新生接待工作（T），
协调 15 人团队、设计活动流程、安排场地物资（A），
活动满意度 95%，获'优秀组织者'称号（R）"
```

### 示例 2：针对性优化

```
用户：我要投产品经理岗位，简历怎么优化？

我会生成：

# 🎯 产品经理简历优化指南

## 关键词提取（JD 分析）

从招聘 JD 中提取关键词：
- 需求分析
- 产品设计
- 原型设计
- 数据分析
- 用户研究
- 项目管理

## 经历改写建议

### 突出相关能力
1. **需求分析能力**
   - "通过用户调研，收集 50+ 份问卷，提炼 3 个核心需求"

2. **产品设计能力**
   - "设计 XX 功能原型，使用 Axure 制作交互原型"

3. **数据分析能力**
   - "分析用户行为数据，发现转化率下降原因，提出优化方案"

4. **项目管理能力**
   - "协调 5 人团队，2 周内完成功能上线"

## 简历结构建议

### 推荐结构
1. 基本信息
2. 教育背景
3. 产品相关经历（重点）
4. 其他经历
5. 技能/作品

### 作品集链接
- 产品分析报告
- 原型设计作品
- 竞品分析报告
```

## 输入格式

```yaml
input:
  target_position: string    # 目标岗位
  experience_level: string   # 经验水平：实习/校招/社招
  industry: string           # 行业：技术/金融/市场等
  has_internship: boolean    # 是否有实习经历
  resume_content: string     # 现有简历内容（可选）
```

## 输出格式

```yaml
output:
  template: string           # 推荐模板
  structure: array           # 简历结构建议
  optimized_content: string  # 优化后的内容
  keywords: array            # 关键词建议
  tips: array                # 注意事项
```
