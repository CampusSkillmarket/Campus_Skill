# 💼 岗位信息获取与匹配助手 (Job Search Assistant)

# 🎯 岗位匹配度分析

## JD 解读

### 岗位要求提取
**硬技能：**
- 本科及以上学历
- 专业不限
- 熟练使用 Office
- 有数据分析能力

**软技能：**
- 沟通能力强
- 学习能力强
- 抗压能力
- 团队合作

**加分项：**
- 有实习经验
- 有相关证书
- 有大厂经历

## 匹配度评估

### 你的情况
请提供：
- 学历专业
- 实习经历
- 技能证书
- 项目经历

### 匹配度计算
| 维度 | 权重 | 你的得分 | 加权得分 |
|------|------|---------|---------|
| 学历 | 20% | - | - |
| 技能 | 30% | - | - |
| 经历 | 30% | - | - |
| 软素质 | 20% | - | - |
| **总计** | 100% | - | **?** |

### 匹配度判断
- 80 分以上：非常匹配，强烈建议投递
- 60-80 分：基本匹配，可以尝试
- 60 分以下：不太匹配，谨慎投递

## 差距分析与提升

### 当前差距
- 缺少 XX 技能
- 缺少 XX 经历

### 快速提升建议
- 1 周内可以学的技能
- 可以包装的经历
- 可以补充的作品
```

# 🔗 内推渠道指南

## 内推平台

### 官方内推
- 公司官网内推入口
- 公司招聘公众号
- 校园招聘官网

### 第三方平台
- 脉脉（职场社交）
- 牛客网（技术岗）
- LinkedIn（外企）
- 内推码小程序

## 人脉拓展

### 学长学姐
**寻找渠道：**
- 校友会
- 校园论坛
- 知乎/脉脉私信

**联系话术：**
> 学长/学姐好，
> 我是 XX 大学 XX 专业的学弟/学妹。
> 看到您在 XX 公司工作，非常敬佩。
> 我对贵公司的 XX 岗位很感兴趣，
> 不知道能否请您帮忙内推？
> 附件是我的简历，感谢您！

### 社交媒体
**LinkedIn 使用：**
1. 完善个人资料
2. 搜索目标公司员工
3. 发送个性化连接请求
4. 建立关系后请求内推

## 内推注意事项

### 内推礼仪
✅ 提前准备好简历
✅ 说明自己的优势
✅ 感谢对方帮助
✅ 及时汇报进展

❌ 不要群发同样消息
❌ 不要催促对方
❌ 不要海投同公司
❌ 不要拿到 offer 就消失

### 内推跟进
- 投递后 3 天询问进展
- 笔试/面试后告知结果
- 无论成败都要感谢
```
