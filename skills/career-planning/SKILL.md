# 🧭 职业规划基础助手 (Career Planning)

## 技能描述

专为大学生和职场新人设计的职业规划 Skill。提供自我认知、行业研究、路径规划、转行指导等服务，帮助用户找到职业发展方向。

## 适用场景

| 细分场景 | 场景描述 |
|----------|----------|
| 自我认知 | 不了解自己适合什么 |
| 行业研究 | 想了解各个行业情况 |
| 路径规划 | 需要规划职业发展路径 |
| 转行准备 | 想转行但不知道如何准备 |

## 核心能力

### 1. 自我认知
- 兴趣测评（霍兰德等）
- 能力评估
- 价值观梳理
- 性格分析（MBTI 等）

### 2. 行业研究
- 行业分类介绍
- 行业发展趋势
- 薪资水平对比
- 入行门槛分析

### 3. 路径规划
- 职业目标设定
- 发展路径设计
- 能力提升计划
- 时间节点规划

### 4. 转行指导
- 转行可行性评估
- 技能差距分析
- 转行路径设计
- 风险规避建议

```
用户：我不知道自己适合什么工作

# 🔍 自我认知指南

## 第一步：兴趣探索

### 霍兰德职业兴趣测试
**6 种类型：**
- R 实际型：喜欢动手操作
- I 研究型：喜欢思考分析
- A 艺术型：喜欢创作表达
- S 社会型：喜欢帮助他人
- E 企业型：喜欢领导影响
- C 常规型：喜欢有序规范

**测试方法：**
1. 列出你喜欢做的事
2. 列出你讨厌做的事
3. 找出兴趣模式
4. 匹配对应职业

### 问题引导
**回答以下问题：**
1. 你平时喜欢做什么？
2. 什么活动让你忘记时间？
3. 你最喜欢什么课程？
4. 你羡慕谁的工作？

## 第二步：能力评估

### 硬技能
- 专业知识
- 技术能力
- 语言能力
- 证书资质

### 软技能
- 沟通能力
- 团队合作
- 解决问题
- 时间管理

### 评估方法
**他人反馈：**
- 问朋友：你觉得我擅长什么？
- 问同事：工作中我表现好的地方？
- 问老师：我有什么天赋？

**成就事件：**
列出你最有成就感的 5 件事
分析其中用到的能力

## 第三步：价值观梳理

### 职业价值观
| 价值观 | 重要性 (1-5) |
|--------|-------------|
| 高薪 | - |
| 稳定 | - |
| 成长 | - |
| 自由 | - |
| 意义 | - |
| 平衡 | - |

### 排序练习
如果只能选 3 个，你选什么？

## 第四步：性格分析

### MBTI 简介
**4 个维度：**
- E/I：外向/内向
- S/N：感觉/直觉
- T/F：思考/情感
- J/P：判断/知觉

**16 种类型对应职业：**
- INTJ：战略家、科学家
- ENFP：记者、心理咨询师
- ISTJ：会计、审计
- ...

## 综合建议

### 职业匹配公式
**适合的工作 = 兴趣 + 能力 + 价值观**

### 下一步行动
1. 做职业兴趣测试
2. 列出能力清单
3. 确定核心价值观
4. 初步筛选职业方向
```

# 🌐 互联网行业分析

## 行业概况

### 行业分类
| 细分领域 | 代表公司 | 特点 |
|----------|----------|------|
| 电商 | 阿里、京东、拼多多 | 成熟稳定 |
| 社交 | 腾讯、微博 | 流量为王 |
| 搜索 | 百度、字节 | 技术驱动 |
| 本地生活 | 美团、饿了么 | 运营密集 |
| 在线教育 | 好未来、猿辅导 | 政策影响 |
| 企业服务 | 钉钉、飞书 | 增长快速 |

## 行业趋势

### 现状分析
**利好因素：**
- 数字化转型加速
- AI 技术带来新机会
- 出海市场广阔

**挑战因素：**
- 流量红利见顶
- 监管趋严
- 竞争加剧

### 未来趋势
1. AI + 各行业应用
2. 产业互联网
3. 出海全球化
4. 硬科技投入

## 薪资水平

### 技术岗（应届）
| 公司梯队 | 薪资范围 |
|----------|----------|
| 一线大厂 | 25-40w |
| 二线大厂 | 18-30w |
| 中型公司 | 12-20w |
| 创业公司 | 8-15w |

### 非技术岗（应届）
| 岗位 | 薪资范围 |
|------|----------|
| 产品 | 15-30w |
| 运营 | 10-20w |
| 市场 | 10-18w |
| 销售 | 8-15w+ 提成 |

## 入行门槛

### 技术岗
- 学历：本科及以上
- 专业：计算机相关优先
- 技能：编程语言、算法
- 项目：GitHub、实习

### 非技术岗
- 学历：本科及以上
- 专业：不限
- 技能：沟通、分析
- 项目：实习、校园经历

## 建议

### 适合人群
✅ 对技术/产品有兴趣
✅ 学习能力强
✅ 抗压能力强
✅ 接受快节奏

### 不建议人群
❌ 追求稳定
❌ 工作生活平衡优先
❌ 不愿持续学习
