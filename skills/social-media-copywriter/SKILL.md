# 社交媒体文案创作专家 (Social Media Copywriter)

专为在小红书、抖音售卖知识的创业大学生设计的文案创作技能。

## 何时使用此技能

- 需要创作小红书知识付费笔记时
- 需要生成抖音知识售卖视频脚本时
- 需要打造个人 IP 人设内容时
- 需要提高内容曝光率和转化率时
- 需要批量生产同主题多平台内容时

## 核心能力

### 1. 小红书爆款文案生成

**标题公式库：**
```
1. 数字 + 结果型："30 天从 0 到 1 万，我的 XX 变现路径全公开"
2. 痛点 + 解决方案型："还在为 XX 发愁？这个方法让我月入 X 万"
3. 对比反差型："同样是大三，为什么我靠 XX 月入 5 万？"
4. 秘密揭露型："99% 的人都不知道的 XX 搞钱路子"
5. 限时限量型："只分享一次！XX 变现的核心秘密"
6. 权威背房型："清华学霸都在用的 XX 学习方法"
7. 失败教训型："踩了 100 个坑后，我总结了 XX 的血泪教训"
8. 速成型："3 分钟学会 XX，小白也能马上上手"
```

**正文结构模板：**
```
【钩子开头】(1-2 句)
- 制造焦虑/共鸣/好奇
- 例："你是不是也..." "有没有人和我一样..."

【人设建立】(2-3 句)
- 身份标签 + 成绩背书
- 例："985 大三 | 靠 XX 实现经济独立 | 已帮助 1000+ 同学"

【干货内容】(主体部分)
- 分点列出，用 emoji 标记
- 每点简短有力，有具体数字
- 例："1️⃣ 第一步：XXX（具体操作）"

【价值升华】(1-2 句)
- 强调改变/结果
- 例："学会这个，你也能..."

【行动号召】(CTA)
- 引导互动/私信/购买
- 例："想要的扣 1" "私信我'XX'获取"
```

**标签策略：**
```
流量标签：#大学生创业 #知识付费 #副业赚钱 #搞钱 #逆袭
垂直标签：#XX 教程 #XX 学习 #XX 变现
人设标签：#95 后创业 #学生党 #自律女孩
热点标签：(根据当下热点添加)
```

### 2. 抖音知识付费脚本生成

**黄金 3 秒开头公式：**
```
1. 提问式："你知道 XX 吗？" "你有没有想过 XX？"
2. 数字式："3 个步骤，让你 XX" "我靠这个赚了 XX 万"
3. 反常识："别再 XX 了！这才是正确方法"
4. 痛点式："是不是总觉得 XX？" "XX 是不是很烦？"
5. 承诺式："看完这条视频，你将学会 XX"
```

**脚本结构模板（60 秒版）：**
```
0-3 秒：钩子开头（吸引注意力）
3-15 秒：痛点放大（制造需求）
15-45 秒：解决方案（核心价值）
45-55 秒：成果展示（建立信任）
55-60 秒：行动号召（引导转化）
```

**转化话术库：**
```
1. 限时优惠："前 100 名只要 XX 元"
2. 稀缺性："只收 X 个学员"
3. 零风险："7 天无理由退款"
4. 从众心理："已经有 XXX 人加入"
5. 价值对比："外面课程卖 XXX，我这里只要 XX"
```

### 3. 人设打造框架

**人设要素：**
```
身份标签：学校/专业/年级/成就
性格特点：励志/专业/亲和/真实
价值主张：帮助什么人解决什么问题
视觉符号：固定穿搭/拍摄风格/封面设计
语言风格：口头禅/表达方式/用词习惯
```

**人设内容比例：**
- 干货内容：60%（建立专业度）
- 生活展示：20%（增加亲和力）
- 成果晒单：10%（建立信任）
- 互动内容：10%（增加粘性）

```

- 目标平台：[小红书/抖音/公众号/B 站]
- 各平台特点：[自动适配]
```

发布前检查：
- [ ] 标题是否有吸引力（数字/痛点/反差）
- [ ] 开头 3 句能否抓住注意力
- [ ] 是否有清晰的人设展示
- [ ] 内容是否分点清晰
- [ ] 是否有具体的数字和案例
- [ ] 是否有行动号召
- [ ] 标签是否覆盖流量词和垂直词

- [ ] 是否有引流路径设计

## 平台特点对比

| 特点 | 小红书 | 抖音 |
|------|--------|------|
| 内容形式 | 图文/短视频 | 短视频 |
| 用户画像 | 年轻女性为主 | 全年龄段 |
| 内容调性 | 精致/有用/真实 | 节奏快/冲击力强 |
| 最佳时长 | 15-60 秒视频 | 30-90 秒视频 |
| 转化路径 | 主页 - 私信 - 微信 | 主页 - 私信 - 微信 |
| 流量逻辑 | 搜索 + 推荐 | 推荐为主 |

## 常见知识付费品类文案要点

### 学习类（英语/考研/考公）
- 强调提分效果和速度
- 用分数/录取结果背书
- 突出方法独特性

### 技能类（PPT/Excel/编程）
- 展示前后对比
- 强调即学即用
- 突出职场价值

### 副业类（自媒体/电商/设计）
- 晒收入截图
- 强调低门槛
- 突出时间自由

### 情感/成长类
- 制造情感共鸣
- 强调认知提升
- 突出改变故事

## 注意事项

1. **合规性**：避免夸大宣传，不承诺具体收益数字
2. **真实性**：人设要真实可信，避免过度包装
3. **持续性**：保持内容更新频率，建议日更
4. **互动性**：及时回复评论私信，建立粉丝关系
5. **差异化**：找到独特定位，避免同质化竞争
