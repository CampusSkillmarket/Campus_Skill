## Word 排版规范

### 毕业论文格式（GB/T 7714）
```
【页面设置】
- 纸张：A4
- 页边距：上 3.5cm，下 2.5cm，左 3cm，右 2.5cm
- 装订线：左侧 1cm

【字体字号】
- 论文题目：黑体小二，居中
- 一级标题：黑体三号
- 二级标题：黑体四号
- 正文：宋体小四，1.5 倍行距
- 英文：Times New Roman

【页眉页脚】
- 页眉：宋体五号，居中
- 页码：底部居中，阿拉伯数字
```

## Excel 常用公式

### 查找引用类
```excel
=VLOOKUP(查找值，查找区域，返回列，精确/模糊)
=XLOOKUP(查找值，查找数组，返回数组)
=INDEX(返回区域，行号，列号)
=MATCH(查找值，查找区域，匹配类型)
```

### 条件计算类
```excel
=SUMIF(条件区域，条件，求和区域)
=COUNTIF(条件区域，条件)
=AVERAGEIF(条件区域，条件，平均区域)
=SUMIFS(求和区域，条件区域 1，条件 1，...)
```

### 文本处理类
```excel
=LEFT(文本，字符数)
=RIGHT(文本，字符数)
=MID(文本，起始位置，字符数)
=TEXT(值，格式代码)
=CONCATENATE(文本 1, 文本 2, ...)
```

### 日期时间类
```excel
=TODAY()          // 今天日期
=NOW()            // 当前时间
=DATEDIF(开始，结束，"Y/M/D")  // 日期差
=WORKDAY(开始，天数，假期)     // 工作日
```

## PPT 设计原则

### CRAP 原则
```
对比 (Contrast)
- 大小对比：标题 36 号，正文 18 号
- 颜色对比：深色背景 + 浅色文字

重复 (Repetition)
- 统一字体
- 统一配色
- 统一图标风格

对齐 (Alignment)
- 左对齐最易读
- 居中对齐最正式
- 避免随意摆放

亲密 (Proximity)
- 相关内容靠近
- 不同内容分开
- 用留白分组
```
