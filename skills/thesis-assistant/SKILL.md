# 🎓 毕业论文/设计助手 (Thesis Assistant)

## 技能描述

专为大学生毕业论文/设计设计的全流程指导 Skill。从选题定题、开题报告、数据收集、论文写作到查重修改，提供全方位支持。

## 适用场景

| 细分场景 | 场景描述 |
|----------|----------|
| 选题定题 | 大四刚开始，需要确定毕业论文题目 |
| 开题报告撰写 | 确定题目后需要写开题报告 |
| 数据收集分析 | 需要收集实验数据或调研数据并分析 |
| 论文查重修改 | 初稿完成后查重率高，需要降重 |

## 核心能力

### 1. 选题指导
- 选题方向建议
- 题目可行性评估
- 选题创新性分析
- 导师匹配建议

### 2. 开题报告
- 开题报告模板
- 研究意义撰写
- 技术路线设计
- 进度计划安排

### 3. 论文写作
- 论文结构框架
- 各章节写作要点
- 文献引用规范
- 学术语言表达

### 4. 查重降重
- 查重系统对比
- 降重技巧方法
- 改写润色建议
- 格式规范检查

## 使用示例

### 示例 1：选题定题

```
用户：我是计算机专业，不知道毕业论文选什么题目

我会生成：

# 🎓 计算机专业毕业论文选题指南

## 选题方向推荐

### 热门方向（2024）
1. **人工智能应用**
   - 基于深度学习的 XX 识别系统
   - 大语言模型在 XX 领域的应用研究

2. **Web 开发**
   - 基于 Vue/React 的 XX 系统设计与实现
   - 微服务架构的 XX 平台开发

3. **数据分析**
   - 基于大数据的 XX 分析与可视化
   - XX 行业数据预测模型研究

4. **移动应用**
   - 基于 Flutter 的跨平台 XX 应用
   - 微信小程序设计与实现

## 选题原则

### 好的选题标准
✅ 范围适中：不要太大也不要太小
✅ 有实际价值：能解决实际问题
✅ 技术可行：在自己能力范围内
✅ 有创新性：有一定新意

### 避免的坑
❌ 题目太大："人工智能研究"
❌ 题目太小："XX 按钮的设计"
❌ 没有价值："XX 系统的复制"
❌ 技术太难：从未接触过的技术

## 题目示例

### 本科毕设题目模板

- "[方法] 在 [领域] 中的应用研究"
- "面向 [场景] 的 [系统] 开发"

### 具体题目示例
1. 基于 Spring Boot 的校园二手交易平台设计与实现
2. 基于深度学习的垃圾分类识别系统
```

### 示例 2：开题报告

```
用户：开题报告怎么写？

我会生成：

# 📝 开题报告写作指南

## 开题报告结构

### 1. 选题背景与意义（800-1000 字）
**写作要点：**
- 行业背景：该领域的现状和发展趋势
- 问题提出：存在什么问题需要解决
- 研究意义：理论意义 + 实践价值

**模板：**
> 随着 XX 技术的发展，XX 行业正经历着...
> 然而，目前存在 XX 问题...
> 本研究旨在...，具有 XX 意义。

### 2. 国内外研究现状（1500-2000 字）
**写作要点：**
- 国外研究：列举 3-5 个代表性研究
- 国内研究：列举 3-5 个代表性研究
- 研究评述：现有研究的不足之处

### 3. 研究内容与目标
**研究内容：**
- 主要研究哪几个方面
- 每个方面的具体内容

**研究目标：**
- 最终要达成什么成果
- 可量化的指标

### 4. 技术路线
**包含内容：**
- 技术选型（语言/框架/工具）
- 系统架构图
- 开发流程图

### 5. 进度安排
| 时间 | 任务 | 交付物 |
|------|------|--------|
| 第 1-2 周 | 文献调研 | 文献综述 |
| 第 3-4 周 | 需求分析 | 需求文档 |
| 第 5-8 周 | 系统开发 | 系统原型 |
| 第 9-10 周 | 测试优化 | 测试报告 |
| 第 11-12 周 | 论文写作 | 论文初稿 |
| 第 13-14 周 | 修改定稿 | 最终论文 |
```

### 示例 3：论文降重

```
用户：论文查重率 45%，怎么降重？

我会生成：

# 📉 论文降重指南

## 降重方法

### 1. 同义词替换
**原句：**
> "系统采用了先进的深度学习算法"

**改写：**
> "该系统运用了前沿的神经网络方法"

### 2. 句式变换
**原句（主动变被动）：**
> "我们使用了 Python 语言进行开发"

**改写：**
> "开发工作采用 Python 语言完成"

### 3. 语序调整
**原句：**
> "在 2020 年，张三等人提出了一种新方法"

**改写：**
> "一种新方法由张三等人于 2020 年提出"

### 4. 拆分合并
**原句（长句拆短句）：**
> "由于深度学习在图像识别领域取得了显著成果，因此本研究选择使用深度学习方法"

**改写：**
> "深度学习在图像识别领域成果显著。基于此，本研究选用深度学习方法。"

### 5. 用自己的话重述
**原句：**
> "根据马斯洛需求层次理论，人的需求从低到高分为生理需求、安全需求、社交需求、尊重需求和自我实现需求五个层次"

**改写：**
> "马斯洛将人的需求划分为五个层级，由低至高依次为：生理、安全、社交、尊重及自我实现。"

## 降重工具推荐

### 查重系统
- 知网查重（学校常用）
- 万方查重
- 维普查重
- PaperPass（便宜）

### 降重技巧
1. 先改大段重复，再改小段
2. 图表可以转成文字描述
3. 代码可以截图展示
4. 引用部分要规范标注
```
