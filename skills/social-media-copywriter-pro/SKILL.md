## 爆款标题公式

### 公式 1：数字 + 结果 + 时间
```
❌ 普通标题：学习文案写作
✅ 爆款标题：3 天学会文案写作，我的学员靠它月入 3 万
```

### 公式 2：痛点 + 解决方案
```
❌ 普通标题：怎么做小红书
✅ 爆款标题：小红书流量差？这 5 个技巧让我 7 天涨粉 1 万
```

### 公式 3：反差 + 好奇
```
❌ 普通标题：我的创业故事
✅ 爆款标题：从挂科到年入百万，我做对了什么？
```

### 公式 4：权威背书 + 稀缺性
```
❌ 普通标题：文案课程推荐
✅ 爆款标题：投资人推荐的文案课，只招 10 人
```

### 公式 5：身份认同 + 行动号召
```
❌ 普通标题：大学生创业
✅ 爆款标题：给想创业的女生一些建议（附实操手册）
```
