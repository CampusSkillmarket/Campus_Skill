# 商业计划书生成器 (Business Plan Generator)

专为创业大学生设计的商业计划书和创业文档创作技能。

## 何时使用此技能

- 需要撰写商业计划书 BP 时
- 需要准备融资路演材料时
- 需要制作财务预测模型时
- 需要撰写市场调研报吿时
- 需要制作竞品分析文档时
- 需要撰写项目可行性报吿时

## 核心能力

### 1. 商业计划书完整框架

**标准 BP 结构（15-20 页）：**

```
1. 封面页 (Cover)
   - 项目名称 + Logo
   - 一句话定位（Slogan）
   - 联系方式

2. 痛点分析 (Problem)
   - 目标用户面临的什么问题
   - 问题的严重程度（数据支撑）
   - 现有解决方案的不足

3. 解决方案 (Solution)
   - 你的产品/服务是什么
   - 如何解决上述问题
   - 核心价值主张

4. 市场规模 (Market Size)
   - TAM（总可服务市场）
   - SAM（可服务市场）
   - SOM（可获得市场）
   - 市场增长趋势

5. 产品介绍 (Product)
   - 产品功能展示
   - 技术架构/服务流程
   - 产品截图/ Demo

6. 商业模式 (Business Model)
   - 如何赚钱
   - 收入来源
   - 定价策略
   - 单位经济模型

7. 竞争优势 (Competitive Advantage)
   - 核心竞争力
   - 护城河
   - 与竞品的差异化

8. 竞品分析 (Competition)
   - 主要竞争对手
   - 竞争格局矩阵
   - 你的优势位置

9. 运营数据 (Traction)
   - 用户数据
   - 收入数据
   - 关键里程碑
   - 增长曲线

10. 营销策略 (Go-to-Market)
    - 获客渠道
    - 营销策略
    - 销售漏斗

11. 团队介绍 (Team)
    - 核心团队成员
    - 背景优势
    - 顾问团队

12. 财务预测 (Financial Projections)
    - 3-5 年收入预测
    - 成本结构
    - 盈亏平衡点
    - 关键财务指标

13. 融资计划 (Funding Ask)
    - 融资金额
    - 资金用途
    - 出让股权
    - 预期达成目标

14. 愿景展望 (Vision)
    - 长期目标
    - 发展规划
    - 社会价值
```

### 2. 投资人关注点清单

**早期项目（天使/Pre-A）：**
```
□ 团队背景（学历、经验、互补性）
□ 市场空间（是否足够大）
□ 产品差异化（是否有独特优势）
□ 早期数据（用户增长、留存）
□ 执行能力（过往成绩）
```

**成长期项目（A 轮及以后）：**
```
□ 收入规模及增速
□ 单位经济模型（LTV/CAC）
□ 市场竞争格局
□ 可扩展性
□ 退出路径
```

### 3. 财务预测模型模板

**收入预测公式：**
```
年收入 = 用户数 × 付费转化率 × 客单价

用户增长 = 上月用户 × (1 + 月增长率)
付费转化率 = 行业基准 × 产品系数
客单价 = 基础价格 × 增值系数
```

**成本结构模板：**
```
固定成本：
- 人力成本（工资 + 社保）
- 办公场地租金
- 服务器/技术成本
- 行政开支

变动成本：
- 获客成本（CAC）
- 服务交付成本
- 支付手续费
- 售后支持
```

**关键财务指标：**
```
毛利率 = (收入 - 直接成本) / 收入
净利率 = 净利润 / 收入
LTV = 客单价 × 毛利率 × 平均生命周期
CAC = 营销费用 / 新增付费用户
LTV/CAC > 3 为健康
```

### 4. 市场分析框架

**PEST 分析：**
```
政治 (Political)：政策环境、监管要求
经济 (Economic)：经济形势、消费能力
社会 (Social)：人口结构、消费习惯
技术 (Technological)：技术趋势、创新机会
```

**波特五力模型：**
```
1. 现有竞争者的竞争程度
2. 潜在进入者的威胁
3. 替代品的威胁
4. 供应商的议价能力
5. 购买者的议价能力
```

### 5. 竞品分析矩阵

**分析维度：**
```
| 维度 | 我们 | 竞品 A | 竞品 B | 竞品 C |
|------|------|--------|--------|--------|
| 价格 | | | | |
| 功能 | | | | |
| 用户体验 | | | | |
| 市场份额 | | | | |
| 融资情况 | | | | |
| 团队规模 | | | | |
```

**差异化定位：**
```
- 价格差异化：高端/性价比
- 功能差异化：人无我有
- 服务差异化：更贴心/更快速
- 渠道差异化：独特获客方式
- 人群差异化：专注细分人群
```

## BP 写作技巧

### 标题/Slogan 公式
```
1. "为 [目标人群] 提供 [核心价值] 的 [产品类型]"
   例："为大学生提供一站式求职辅导的在线平台"

2. "[差异化特点] 的 [品类名]"
   例："更懂年轻人的第一理财 APP"

3. "用 [技术/方法] 解决 [痛点问题]"
   例："用 AI 帮你 3 分钟生成专业简历"
```

### 痛点描述技巧
```
1. 场景化："每天花 3 小时找资料，效率低下"
2. 数据化："80% 的大学生求职时没有专业指导"
3. 情感化："明明很优秀，却因为简历被刷掉"
4. 对比化："别人拿到 offer 了，你还在海投"
```

### 市场规模计算
```
自上而下法：
- 全国大学生 4000 万 × 求职辅导渗透率 5% × 客单价 500 元 = 100 亿

自下而上法：
- 单月获客 500 人 × 12 月 × 客单价 500 元 × 3 年 = 9000 万
```

### 竞争优势表达
```
1. 先发优势："最早进入 XX 细分领域"
2. 技术优势："独家 XX 算法，效率提升 10 倍"
3. 资源优势："与 XX 家高校建立独家合作"
4. 团队优势："核心团队来自 XX，有 XX 年经验"
5. 数据优势："积累 XX 万用户行为数据"
```

## 常见创业品类 BP 要点

### 教育科技类
- 强调市场规模（教育刚需）
- 突出提分/学习效果数据
- 展示获客渠道和成本
- 说明复购和转介绍率

### 电商/新零售类
- 展示供应链优势
- 强调复购率和客单价
- 说明库存周转效率
- 突出差异化选品

### SaaS/工具类
- 强调 ARR/MRR 增长
- 展示留存率（NDR）
- 说明获客效率（LTV/CAC）
- 突出产品技术壁垒

### 本地生活类
- 强调区域密度优势
- 展示单店/单点盈利模型
- 说明可复制性
- 突出运营效率

## 融资策略建议

### 融资节奏
```
天使轮：想法/原型 → 50-200 万 → 验证 PMF
Pre-A 轮：早期数据 → 300-800 万 → 验证模式
A 轮：规模收入 → 1000-5000 万 → 快速扩张
```

### 投资人类型
```
1. 天使投资人：决策快、金额小、看团队
2. 天使机构：专业化、有资源、看数据
3. VC 机构：金额大、要求高、看增长
4. 产业资本：有资源协同、看战略价值
```

### BP 投递策略
```
1. 优先投专注早期/赛道的投资人
2. 通过熟人引荐成功率更高
3. 准备 1 页版/3 页版/完整版多种格式
4. 附上产品 Demo/数据截图增加可信度
```

## 注意事项

1. **数据真实性**：不要夸大或虚构数据
2. **逻辑自洽**：各部分论述要一致
3. **简洁清晰**：每页只讲一个核心点
4. **视觉呈现**：多用图表，少用文字
5. **版本管理**：根据反馈持续迭代
6. **保密意识**：敏感信息注意保护
