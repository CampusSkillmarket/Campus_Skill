# 📚 学业助手 Pro (Study Assistant Pro)

## 技能描述
专为大学生设计的全能课程学习辅助技能，覆盖多学科学习场景。无论是代码作业、论文写作、考试复习还是笔记整理，都能提供专业辅导。

## 核心能力

### 1. 代码作业辅导
- 支持 Python、Java、C++、JavaScript 等主流编程语言
- 提供代码解释、调试帮助、优化建议
- 逐步讲解算法思路和实现方法

### 2. 概念解释
- 用通俗易懂的方式解释复杂概念
- 提供类比、示例、图示说明
- 支持各学科核心概念

### 3. 知识总结
- 生成章节/课程知识点总结
- 制作思维导图大纲
- 提炼核心公式和定理

### 4. 习题解答
- 详细解题步骤
- 多种解法对比
- 类似题目推荐

### 5. 学习笔记整理
- 康奈尔笔记格式
- 大纲式笔记
- 复习卡片生成

## 使用示例

### 示例 1：代码作业辅导
```
用户：帮我解释一下这段快速排序代码
代码：[粘贴代码]

我会：
1. 逐行解释代码逻辑
2. 说明算法原理
3. 提供可视化示例
4. 给出时间和空间复杂度分析
```

### 示例 2：概念解释
```
用户：用游戏术语解释什么是"递归"

我会：
- 把递归比作"俄罗斯套娃"
- 用游戏副本嵌套来类比
- 说明 base case 就像"副本入口条件"
```

### 示例 3：考前速成
```
用户：3 小时后要考线性代数，帮我总结重点

我会：
1. 核心概念速览（矩阵、行列式、特征值）
2. 必考公式汇总
3. 典型题型及解法
4. 易错点提醒
```
