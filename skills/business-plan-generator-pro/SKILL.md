# 💼 商业计划书生成器 (Business Plan Generator)

## 技能描述
专为创业大学生设计的商业计划书和创业文档创作技能。擅长打造专业级 BP、财务预测模型、市场分析报告，帮助用户获得投资人青睐。

## 核心能力

### 1. 商业计划书生成
- 完整 BP 结构（10-15 页标准版）
- 精简版 BP（5 页路演版）
- 详细版 BP（30 页完整版）

### 2. 财务预测模型
- 3-5 年财务预测
- 收支平衡分析
- 现金流预测
- 关键财务指标计算

### 3. 市场分析报告
- 市场规模估算（TAM/SAM/SOM）
- 竞品分析
- 用户画像
- 行业趋势分析

### 4. 投资人路演材料
- 路演 PPT 大纲
- 投资人 Q&A 准备
- 估值建议
- 融资策略

## BP 标准结构

```
1. 执行摘要 (Executive Summary)
2. 问题与痛点 (Problem)
3. 解决方案 (Solution)
4. 产品介绍 (Product)
5. 市场分析 (Market Analysis)
6. 商业模式 (Business Model)
7. 竞争分析 (Competition)
8. 营销策略 (Go-to-Market)
9. 团队介绍 (Team)
10. 财务预测 (Financial Projections)
11. 融资需求 (Ask)
12. 附录 (Appendix)
```

【解决方案】
- 核心产品：[产品名称]

【商业模式】

【财务亮点】
- 第 1 年营收：[预测]

【融资需求】

- 出让股份：[百分比]

### 市场分析模板
```
## 市场规模

### TAM (总可服务市场)
- 全国大学生：4000 万
- 求职辅导需求：30%
- 客单价：500 元
- TAM = 4000 万 × 30% × 500 = 600 亿

### SAM (可服务市场)
- 目标城市：一线 + 新一线
- 覆盖大学生：1000 万
- SAM = 1000 万 × 30% × 500 = 150 亿

### SOM (可获得市场)
- 第 3 年目标市场份额：1%
- SOM = 150 亿 × 1% = 1.5 亿
```

## 常见创业品类 BP 要点

### 电商平台类
- 供应链优势
- 获客成本 (CAC)
- 用户生命周期价值 (LTV)
- 复购率

### SaaS 工具类
- 月经常性收入 (MRR)
- 客户流失率 (Churn)
- 获客回收周期 (Payback)
- 净推荐值 (NPS)

### 内容平台类
- 日活/月活 (DAU/MAU)
- 用户时长
- 内容生产成本
- 商业化路径

### 硬件创业类
- 单机成本结构
- 量产能力
- 渠道布局
- 售后体系

## 投资人关注点

### 早期项目（天使/Pre-A）
1. 团队背景（大厂/创业经历）
2. 市场空间（是否足够大）
3. 产品差异化（竞争优势）
4. 早期数据（增长趋势）

### 成长期项目（A 轮及以后）
1. 营收规模及增速
2. 单位经济模型 (UE)
3. 可复制性
4. 竞争壁垒

## 融资建议

### 估值方法
- 市场法：参考同类公司
- 收益法：未来现金流折现
- 成本法：投入成本加成

### 融资节奏
- 天使轮：想法 + 团队 → 50-200 万
- Pre-A 轮：MVP+ 早期数据 → 300-500 万
- A 轮：验证模式 + 规模增长 → 1000-3000 万

### 投资人类型
- 天使投资人：决策快、金额小
- VC 机构：金额大、要求高
- 产业资本：资源多、战略协同

## 使用提示

1. **提供项目信息**：品类、模式、阶段
2. **说明融资需求**：金额、用途、预期
3. **指定 BP 类型**：精简版/标准版/详细版
4. **补充团队信息**：核心成员背景

## 注意事项

- 财务预测需合理，不要过度乐观
- 突出差异化竞争优势
- 数据要有来源支撑
- 根据投资人偏好调整重点
