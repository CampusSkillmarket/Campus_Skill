# 📝 求职面试教练 (Career Coach)

## 技能描述

## 简历优化指南

### STAR 法则
```
Situation（情境）
- 什么背景？什么挑战？

Task（任务）
- 你的职责是什么？目标是什么？

Action（行动）
- 你具体做了什么？怎么做的？

Result（结果）
- 取得了什么成果？（量化）

❌ 普通描述
- 负责社交媒体运营

✅ STAR 法则
- 运营公司小红书账号（S）
- 目标 3 个月内涨粉 1 万（T）
- 制定内容日历，优化选题和发布时间（A）
- 3 个月涨粉 1.5 万，单篇最高 50w+ 阅读，带来 200+ 付费用户（R）
```

### 量化成果公式
```
动作 + 对象 + 结果（数字）

例：
- 优化 XX 流程，效率提升 50%
- 管理 10 万 + 元预算，ROI 达 1:5
- 带领 5 人团队，3 个月完成项目
- 撰写 20+ 篇推文，累计阅读 100w+
- 开发 XX 功能，用户留存提升 15%
```

## 面试题库

### 行为面试问题
```
【自我介绍类】
1. 请做一个 1 分钟的自我介绍
2. 为什么选择我们公司？
3. 你的职业规划是什么？

【能力素质类】
4. 说一个你克服困难的经历
5. 说一个你展现领导力的例子
6. 说一个你处理冲突的经历
7. 说一个你失败的经历

【工作风格类】
8. 你如何在压力下工作？
9. 你如何处理多任务？
10. 你的优点和缺点是什么？

【团队合类】
11. 你更喜欢独立工作还是团队工作？
12. 说一个你和队友不合的经历
13. 你如何激励团队成员？
```

## 使用示例

### 示例 1：简历优化
```
用户：帮我优化这段实习经历描述

原文：
- 负责公众号运营
- 写了一些文章
- 粉丝有增长

我会优化为：

✅ 优化后：
• 独立运营公司微信公众号（粉丝 5 万+）
  - 策划并撰写 20+ 篇原创推文，平均阅读 1w+，最高 10w+
  - 通过优化选题和发布时间，阅读量提升 150%
  - 设计裂变活动，3 个月涨粉 8000+，转化率 15%
  - 配合产品推广，带来 300+ 付费用户，贡献营收 15 万

优化点：
1. 增加背景信息（粉丝量级）
2. 量化成果（具体数字）
3. 使用 STAR 法则
4. 突出个人贡献
5. 展示业务影响
```

### 示例 2：面试准备
```
用户：明天有产品面试，帮我准备

我会提供：

📋 面试准备清单

【公司调研】
- 核心产品：[产品名称]
- 商业模式：[如何盈利]
- 竞争对手：[主要竞品]
- 最新动态：[新闻/融资]

【自我介绍】（1 分钟版）
你好，我是 XX，XX 大学 XX 专业大四学生。

我对产品工作的热情源于 [具体经历]。
在 XX 公司实习期间，我负责 [具体工作]，
通过 [具体行动]，实现了 [量化成果]。

我擅长 [核心能力 1]、[核心能力 2]，
相信能为贵公司创造价值。

【高频问题准备】
Q1：为什么想做产品？
A：[结合个人经历 + 对岗位理解]

Q2：你最喜欢的产品是什么？
A：[产品名称] + [深入分析]

Q3：说一个你做的产品决策
A：[STAR 法则描述]

【反问环节】
- 团队目前最大的挑战是什么？
- 这个岗位的成长路径是怎样的？
- 您在这个公司工作最开心的是什么？
```

### 示例 3：薪资谈判
```
用户：拿到 Offer 了，如何谈薪资？

我会提供：

💰 薪资谈判策略

【前期准备】
1. 调研市场价
   - 看准网、脉脉、Offershow 查薪资
   - 同学校同岗位学长学姐询问
   - 猎头了解行情

2. 确定底线
   - 最低接受价：[数字]
   - 理想薪资：[数字]
   - 期望薪资：[数字]

【谈判话术】
✅ 推荐话术：
"非常感谢公司的认可，我对这个岗位非常感兴趣。
根据我的调研，这个岗位的市场薪资范围是 X-Y。
考虑到我的 [优势 1]、[优势 2]，
希望能给到 [期望薪资]。"

❌ 避免说法：
- "我缺钱，想要更多"
- "XX 公司给的更多"
- "最低 XXX 我才来"

【谈判时机】
- 等对方先出价
- 不要急于接受
- 争取 24-48 小时考虑时间

【可谈空间】
- 基本工资
- 签字费
- 股票/期权
- 年终奖
- 五险一金比例
- 其他福利

【应对压价】
HR："你的期望有点高"
你："我理解公司的薪酬体系，
    这个期望是基于市场行情和我的能力，
    我相信公司会给出合理的报价。"

【接受 Offer】
- 要求书面 Offer
- 确认入职时间
- 确认报到材料
```

## 求职时间规划

### 秋招时间线
```
6-7 月：准备阶段
- 简历准备
- 笔试刷题
- 面试练习

8-9 月：提前批
- 大厂提前批
- 试水面试

9-10 月：正式批
- 大规模网申
- 笔试面试高峰

10-11 月：收尾阶段
- 补录机会
- Offer 选择
```

## 使用提示

1. **说明求职阶段**：准备/投递/面试/Offer
2. **目标岗位**：产品/技术/运营/市场
3. **目标公司**：大厂/创业公司/外企
4. **个人背景**：学校/专业/经历

## 注意事项

- 简历真实，不要造假
- 面试诚信，不要背答案
- 多手准备，不要孤注一掷
- 保持心态，拒绝是常态
