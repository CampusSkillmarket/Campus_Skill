# 🏠 宿舍关系处理指南

## 问题分析

### 问题类型
- 噪音问题：打游戏、打电话、晚睡
- 卫生问题：不打扫、乱扔垃圾
- 作息问题：早睡 vs 晚睡
- 用品问题：乱用东西

## 沟通策略

### 第一步：私下沟通
**错误方式：**
❌ "你能不能别吵了！"
❌ "你太自私了！"

**正确方式：**
✅ "我最近睡眠不太好，能不能 11 点后小声一点？"
✅ "我明天有早课，能不能麻烦你戴耳机？"

### 第二步：集体商量
如果私下沟通无效：
- 召开宿舍会议
- 制定宿舍公约
- 明确作息时间
- 分工打扫卫生

### 第三步：寻求外部帮助
如果还是无效：
- 找辅导员协调
- 申请换宿舍
- 校外租房

## 宿舍公约模板

```

1. 作息时间：23:00 后保持安静

4. 用品使用：先问再用
5. 空调温度：协商确定

日期：_______
```

## 心态调整

### 理解包容

- 相互理解，相互包容
- 求同存异

### 设立边界
- 自己的权利要维护
- 合理要求要表达
- 过分要求要拒绝

# 🤝 职场人脉建立指南

## 人脉类型

### 内部人脉
- 直属领导
- 团队成员
- 跨部门同事
- 上级领导

### 外部人脉
- 行业同行
- 客户资源
- 供应商
- 校友资源

## 建立方法

### 入职初期（1-3 月）
**目标：** 认识团队，建立印象

**行动：**
1. 记住每个人的名字和职位
2. 主动打招呼，微笑
3. 参加集体活动
4. 虚心请教，多学习

### 成长期（3-12 月）
**目标：** 深化关系，建立信任

**行动：**
1. 工作靠谱，说到做到
2. 主动帮助他人
3. 分享有价值的信息
4. 参加行业活动

### 成熟期（1 年 +）
**目标：** 互惠互利，长期经营

**行动：**
1. 资源互换
2. 推荐机会
3. 定期联系
4. 维护关系

## 社交技巧

### 破冰话题
- 工作相关："你是哪个部门的？"
- 共同经历："你也是 XX 大学毕业的？"
- 兴趣爱好："你平时喜欢做什么？"
- 热点话题："最近那个新闻你看了吗？"

### 加微信后
**第一条消息：**
> "XX 你好，我是 XX 部门的 XXX，
> 今天很高兴认识你，
> 以后多交流！"

**朋友圈互动：**
- 点赞是基本礼貌
- 评论要真诚
- 不要只发广告

## 人脉维护

### 定期联系
- 每月至少联系一次
- 节日问候
- 有好机会想到对方

### 价值提供
- 分享行业信息
- 推荐合适人选
- 介绍业务机会

### 记录管理
建立人脉表格：
| 姓名 | 公司 | 职位 | 认识时间 | 最近联系 | 备注 |
|------|------|------|----------|----------|------|
