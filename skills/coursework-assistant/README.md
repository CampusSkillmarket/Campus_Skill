## 📖 技能介绍

本 Skill 专为学习场景设计，提供专业的 AI 辅助功能。

## 📥 下载方式

### 方式一：在线查看（推荐）

1. 点击 [SKILL.md](https://github.com/CampusSkillmarket/skillhub-skills/blob/main/skills/coursework-assistant/SKILL.md) 查看完整内容
2. 复制全部内容
3. 粘贴到你的 AI 助手配置中

### 方式二：下载文件

1. 下载本仓库的 ZIP 文件
2. 解压后找到 `skills/coursework-assistant/SKILL.md`
3. 复制内容到 AI 助手配置

### 方式三：直接克隆

```bash
git clone https://github.com/CampusSkillmarket/skillhub-skills.git
cd skillhub-skills/skills/coursework-assistant
```

## ⚡ 快速开始

1. **复制 Skill 内容**
   - 打开 [SKILL.md](https://github.com/CampusSkillmarket/skillhub-skills/blob/main/skills/coursework-assistant/SKILL.md)
   - 全选并复制所有内容

2. **配置到 AI 助手**
   - 打开你的 AI 助手配置界面
   - 找到 Skill/技能配置区域
   - 粘贴内容并保存

3. **开始使用**
   - 按照 SKILL.md 中的说明使用
