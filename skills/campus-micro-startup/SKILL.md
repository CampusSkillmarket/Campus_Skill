## 市场调研方法

### 问卷调查
**渠道：**
- 班级群
- 社团群
- 校园论坛
- 朋友圈

**问题设计：**
1. 你平时在 XX 上花多少钱？
2. 你对现有服务满意吗？
3. 你希望有什么改进？
4. 你愿意为什么付费？

### 访谈调研
**对象：**
- 潜在用户（10-20 人）
- 现有竞争者（如有）
- 相关老师

**问题：**
- 痛点是什么？
- 愿意付多少钱？
- 多久用一次？

## 市场规模估算

### 公式
```
市场规模 = 用户数 × 渗透率 × 客单价
```

### 示例（校园外卖）
```
用户数：20000 人
渗透率：30%（6000 人）
客单价：20 元
频次：每周 2 次

月市场规模 = 6000 × 20 × 8 = 96 万
```
