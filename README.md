> 32 个即插即用的 AI 技能，帮你搞定学习、求职、创业

## 使用方法

1. **找技能** → 从下方列表选一个
2. **复制内容** → 打开 Skill 文件夹里的 `SKILL.md`，全选复制
3. **粘贴使用** → 粘贴到你的 AI 助手（Qwen Code / Claude / ChatGPT 等）
