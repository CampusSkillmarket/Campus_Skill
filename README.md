# SkillHub AI Skills

> 32 个即插即用的 AI 技能，帮你搞定学习、求职、创业

## 这是什么？

这是一个为大学生和年轻创业者打造的 AI 技能库。每个 Skill 都是一段精心设计的提示词，配置到你的 AI 助手后，就能立刻获得一个懂你的专业助手。

**两种用法：**
- **自己用** → 解决学习、求职、生活中的实际问题
- **创业用** → 基于 Skill 二次开发，打造 AI 产品服务校园市场

## 快速开始

1. **找技能** → 从下方场景选一个你需要的
2. **复制内容** → 打开 Skill 文件夹里的 `SKILL.md`，全选复制
3. **粘贴使用** → 粘贴到你的 AI 助手（Qwen Code / Claude / ChatGPT 等）
